# My Blog Full-Stack Project

This is a full-stack blog project with:

- **Backend:** Spring Boot (Java 17)
- **Frontend:** Angular
- **Database:** PostgreSQL 
- **No sudo required** for setup

---

## Prerequisites

- Docker 
- Node.js and npm (installed via `nvm`)
- Java 17+
- Maven
